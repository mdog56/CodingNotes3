Git Notes

Command Description
ls Lists the current directory contents
pwd Prints the working directory
cd Change directory
cd .. Go up one directory
cd ../.. Go up two directories
cd ../foldername Go up one directory, then down into a defined directory
mkdir Create a directory
touch Create a file
tab Auto-completes your command
clear Clears your terminal
echo "" Prints text or string data into the terminal
cat Reads a file sequentially and prints it to the standard output
↑ Goes up
↓ Goes down
> Redirects the standard output of the command on the left to thefile on the right
>> Adds the standard output of the command on the left to the file on the right

Category Code Description
Selector nameOfTag Type selector is the same as the HTML tag (* selects all elements)
Selector .ValueOfClassAttribute Selects all elements of a common class
Selector #NameOfId Selects the element with that ID value
Selector [NameOfAttribute] Selects all the elements with a certain Attribute
Selector type[attribute*="value" Selects all the elements of a certain type where the attribute contains any instance of the given value
Selector element1>element2 Selects all the element2 elements who's parents are element1 elements
Selector element1+element2 Selects the first element2 element that is placed immediately after element1 elements
Rule &font-face Specifies a custom font with which to display text
Pseudo-element element::before Insert something before the content of each element.
Pseudo-element element::after Insert something after the content of each element
Pseudo-class Selector element:link Selects all unvisited links. Set pseudo-class rules in the proper order so that the styles cascade correctly
Pseudo-class Selector element:visited Selects that element if it's been visited
Pseudo-class Selector element:hover Selects that element when it is hovered over
Pseudo-class Selector element:active Selects that element when it's clicked on
Pseudo-class Selector element:focus Selects that elements that has focus
Pseudo-class Selector element:disabled Selects that elements that's disabled
Pseudo-class Selector :not(element) Selects all elements that don't match the given selector
